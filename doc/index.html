Image uploader

Example

            var uploader = new ImageUploader('[data-js="uploader"]', {
                maxItems: 2
            });
            uploader.on('adding', function(file, done, progress) {
                progress(10);
                var croppedFile = crop(file, 200, 200);
                progress(50);
                var picture = {
                    url: croppedFile
                };
                progress(100);
                done(picture);
            });
            uploader.add({
                url: 'http://example.com/picture.jpg'
            });

Options

maxItems
int Max number of images

Methods

            // example of picture object
            // url can be url or image in base64
            var picture = {
                url: 'http://example.com/picture.jpg'
            };
add(picture)
Add picture
update(index, picture)
Update picture at index
delete(index)
Remove picture at index
on(event, callable)
Bind function to an event

Events

adding
Call before add
updating
Call before update
deleting
Call before delete
